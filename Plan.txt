Module 1: Xây dựng nền tảng (Infrastructure) & Data Lake
Buổi 1: Khởi tạo thế giới ảo với Docker.

Học: Khái niệm Containerization (tại sao DE lại cần Docker?).

Làm: Cài Docker. Viết file docker-compose.yml khởi tạo PostgreSQL (chứa dữ liệu đích) và MinIO (mô phỏng Amazon S3).

Buổi 2: Làm quen Object Storage.

Học: Khác biệt giữa File System và Object Storage (Bucket, Object).

Làm: Tạo Bucket trên MinIO. Tải dataset Olist E-commerce từ Kaggle về máy.

Buổi 3: Python Data Ingestion (Bronze Layer).

Học: Cách dùng thư viện boto3 (AWS SDK) trong Python.

Làm: Viết script Python tự động đọc file CSV từ máy và upload lên bucket bronze-layer trên MinIO.

Module 2: Trái tim của DE - Xử lý dữ liệu với PySpark
Buổi 4: Hello PySpark.

Học: Kiến trúc phân tán của Spark (Driver, Executor), tại sao Spark đập chết Pandas khi xử lý Big Data.

Làm: Setup môi trường PySpark (Jupyter Notebook). Đọc file CSV trực tiếp từ MinIO bronze-layer vào Spark DataFrame.

Buổi 5 & 6: Data Transformation (Silver Layer).

Học: Các hàm xử lý dữ liệu cơ bản trong Spark: Filter, DropNulls, Cast Type, Join.

Làm: Làm sạch bảng Orders, Items, Customers. Chuyển định dạng timestamp, loại bỏ đơn hàng bị hủy.

Buổi 7: Lưu trữ tối ưu hóa.

Học: Cấu trúc file Parquet vs CSV (Columnar vs Row-based).

Làm: Ghi toàn bộ dữ liệu đã làm sạch từ Spark xuống bucket silver-layer của MinIO dưới định dạng Parquet.

Module 3: Mô hình hóa dữ liệu (Data Modeling) & Data Warehouse
Buổi 8: Tư duy thiết kế kho dữ liệu.

Học: Kiến trúc Star Schema (Fact table và Dimension table là gì?).

Làm: Vẽ nháp trên giấy hoặc draw.io sơ đồ liên kết giữa Fact_Sales với Dim_Customers, Dim_Products.

Buổi 9: Tạo bảng trong DWH.

Học: DDL (Data Definition Language) trong SQL.

Làm: Dùng DBeaver/DataGrip kết nối vào PostgreSQL. Chạy lệnh CREATE TABLE tạo cấu trúc Star Schema đã vẽ.

Buổi 10 & 11: Transform & Load (Gold Layer).

Học: Xử lý logic nghiệp vụ bằng code.

Làm: Viết script PySpark đọc dữ liệu Parquet từ silver-layer, join lại theo thiết kế Star Schema và dùng phương thức write.jdbc đẩy thẳng dữ liệu vào PostgreSQL.

Module 4: Tự động hóa với Apache Airflow
Buổi 12: Setup Airflow.

Học: DAG (Directed Acyclic Graph) là gì? Task, Operator là gì?

Làm: Thêm Airflow vào file docker-compose.yml ở buổi 1. Khởi động giao diện web của Airflow.

Buổi 13 & 14: Biến script thành Pipeline.

Học: Tư duy Dependency (task nào chạy trước, task nào chạy sau).

Làm: Viết DAG gom tất cả các file Python/Spark từ Buổi 3 đến Buổi 11 lại. Lập lịch cho pipeline chạy tự động mỗi ngày (ví dụ: schedule_interval='@daily').

Module 5: Trình bày CV & "Flex" kiến thức
Buổi 15: Trực quan hóa (BI).

Làm: Thêm tool BI Metabase vào Docker. Kết nối với PostgreSQL, kéo thả 2 chart đơn giản (Ví dụ: Tổng doanh thu theo tháng). Bạn đã có kinh nghiệm làm Funnel Analysis ở Mcredit nên phần này sẽ rất dễ.

Buổi 16: Đóng gói Project.

Làm: Đưa code lên GitHub. Viết file README.md theo chuẩn: Introduction -> Architecture Diagram -> Prerequisites -> How to run (chỉ với 1 lệnh docker-compose up).
